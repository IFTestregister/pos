Was versteht man unter dem Begriff „Informatik“?​

Wissenschaft & Anwendung von der systematischen, maschinell unterstützten:

  • Verarbeitung,
  • Speicherung,
  • Übermittlung

von Information

Definition = Information + Automatik

Basiert auf Mathematik und Elektrotechnik

Was versteht man unter dem Begriff „Information“?​

Sequenz (Folge) von Symbolen.

Was ist ein Symbol?​

  • Objekt
  • Bild
  • Wort
  • Klang

Beispiele: Stoppschild, "Top Secret", Klingelton

Was bedeutet Kodierung?​

Regelung für das Abbilden von Information

Was bedeutet Zeichenkodierung?​

Symbole werden auf Bitfolgen (Zeichenfolgen) abgebildet => zum auf PC

  • Speichern,
  • Weiterverarbeiten,
  • Ausgeben

Arten:

  • ASCII (American Standard Code for Information Interchange)
  • ISO-8859 (International Standards Organisation)
    • ISO-8859-1 westeuropäische Zeichen
    • ISO-8859-15 inkl. €-Zeichen

Was sind Daten?​

analoge Daten​

  • Thermometer (Quecksilberhöhe)
  • Uhren (mittels Zeiger)
  • Musik (Speicherung auf Schallplatten)

digitale Daten​

Computer- lesbare und -bearbeitbare, in der Regel digitale Repräsentation von Information.

Information wird in Zeichenketten kodiert, mit strengen Regeln (Syntax)

Formatierte Daten​

Formatierte digitale in einem festen Aufbau aufgezeichnete Daten (besonders geeignet für Interpretation)

Was ist ein Computer?​

Rechner, Hardware, materielle Komponenten eines Informationsverarbeitungssystems

Was ist eine Software?​

die nicht materiellen Komponenten eines Informationsverarbeitungssystems:

  • Programme
  • Daten

Was ist ein Algorithmus?​

Eindeutige Beschreibung eines Problemlösungsverfahrens:

  • Kaffee kochen

Was ist ein Programm?​

Software die Anweisungen enthält (Umsetzung von Algorithmen)

vor dem übersetzen​

in einer Programmiersprache formulierte Algorithmen Sequenz von Anweisungen in einer Programmiersprache

nach dem übersetzen​

Sequenz von Instruktionen in einer Maschinensprache

Was ist die Maschinensprache?​

Anweisungen (Folgen aus 1 & 0) die der Prozessor direkt versteht

Was ist die Programmiersprache?​

Formale Sprache um dem Computer Anweisungen zu geben.

  • Assembler
  • C, C++
  • Python, Java, C#, Ruby
  • Smalltalk, Lisp, Prolog, Haskell

Was versteht man unter Syntax?​

geregeltes System mit Ausdrücken aus einem grundlegen Zeichensatz Regelverletzung = Syntaxfehler

Was ist ein Compiler?​

Computerprogramm zur Umwandlung von einem Programm von einer Quellsprache in eine Zielsprache (= Assembler oder Maschinensprache)

Was ist ein Interpreter?​

Computerprogramm zur Analyse und Ausführung des Quellcodes (während das Programm läuft)

Was bedeutet „Shell“?​

Zwischenschicht zwischen Programm und Betriebssystem oder Interpreter => Prozess (kann mit CTRL-C beendet werden)

Was bedeutet Prozess?​

Gestartetes Programm

Was ist ein Texteditor?​

Programm zur Textbearbeitung

Text wird in einer Datei gespeichert/von einer Datei ausgelesen Dateien werden in einem Verzeichis gespeichert

Was bedeutet „IDE“?​

Integrated Development Environment

Integrierte Entwicklungsumgebung, Komponenten:

  • Texteditor
  • Compiler oder Interpreter
  • Debugger

Was bedeutet „EVA“?​

Eingabe-Verarbeitung-Ausgabe

Was ist eine Gleitkommazahl?​

Zahl mit Kommastellen

Was ist ein Modul?​

abgeschlossene Einheit einer Software, bestehend aus Code und Daten.

Was ist eine Anweisung?​

Ausführbarer Befehl in einer Programmiersprache => Viele Anweisungen = Programm

Anweisungen enthalten Ausdrücke oder andere Anweisungen

wird ein Funktionsaufruf alleine angeschrieben, handelt es sich um eine Aufrufanweisung (call statement)

die Trennung von Anweisungen erfolgt durch ';'

Was ist ein Keyword?​

Jede Programmiersprache hat reservierte Ausdrücke, die Keywords

Python:

  • if
  • from
  • import

Was ist ein Muster?​

bewährte Lösungsschablonen für Probleme

Was ist ein Objekt?​

Alles in Python.

Python überprüft immer den Typ eines Objektes mit 'type'

Jedes Objekt hat

  • einen Typ
  • eine eindeutige Identität
  • einen aktuellen Zustand
  • ein Verhalten

Was bedeutet Garbage Collection?​

ist ein Objekt keiner Variable mehr zugewiesen, dann ist das Objekt Müll und wird vom Garbage Collector von Python erkannt und aus dem Speicher gelöscht
